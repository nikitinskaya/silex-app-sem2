# A php project for the 2nd semester webdev course

## Install and run:
`composer install`

`php -S localhost:port`

## Check out the [heroku app](https://damp-journey-53796.herokuapp.com/)

`/hello/yourname`

`/author`

`/info`

`/print`

`/rates`
